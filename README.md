## 安装

```
$ yarn
```

## 本地开发

```
$ yarn start
```

这个命令会开启一个服务，并自动打开浏览器。当有文件更改时页面会自动刷新。

## 构建

```
$ yarn build
```

这个命令会生成静态文件到 `build` 目录。

## 部署

```
$ GIT_USER=<Your GitHub username> USE_SSH=true yarn deploy
```

如果需要部署到 GitHub pages，这个命令会先构建静态页面，然后推送到 `gh-pages` 分支。
